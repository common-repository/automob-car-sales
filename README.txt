The easiest way to sell vehicles with WordPress.

== Description ==

Automob Car Sales is a complete vehicle dealer solution for selling vehicles in a light, easy and performant way.

Some features of the plugin include:

* Powerful AJAX search filtering for your inventory. Place the [automob_vehicle_search_form] on any wordpress page to output a very powerful search form, which is able to filter vehicles without refreshing the page.
* Quickly add/remove vehicles, upload photos & change prices.
* Car details pages (with a contact form for each vehicle)
* User friendly Admin panel with with automatic saving of individual fields when editing.
* Drop your vehicle inventory onto any page using the [automob_vehicle_inventory] shortcode.
* Beautiful ribbons
* Image gallery for each vehicle
* Integrates well with all WordPress themes
* Bundled products


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/automob` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create a wordpress page for your search form e.g Home
4. Add the shortcode [automob_vehicle_search_form] in the editor and click "Update" to save.
5. Create another wordpress page for your inventory/showroom page.
6. Add the [automob_vehicle_inventory] in the editor and click "Update" to save.
7. Go to Car Listings->Automob Settings screen. select the inventory page you created above, under "Search results page".
8. Click save changes
9. If you browser the page with the [automob_vehicle_search_form] shortcode, you should see a search form which on submission, redirects to a showroom/inventory page with results.


== Frequently Asked Questions ==

= What PHP version is required for this plugin? =
This plugin requires a minimum of PHP 5.3.

= How do I list the vehicles on my website? =
Create a page e.g called 'Showroom', that contains the shortcode [automob_vehicle_inventory]. Browse to that page a see the listing.
